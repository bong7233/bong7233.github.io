단순회귀

  • 종속변수 : 결과, 타겟, 레이블 (y)
  • 독립변수 : 변수 (x)
  • 회귀선의 공식 $\hat{y}$ = b0 + b1$x$
    • 회귀선의 공식을 도출과정 중요
  • Tabular Data
    • 행데이터 : 관측치 (열 별로 데이터타입이 다를수있음)
    • 열데이터 : 변수 (같은 열에는 같은 데이터타입)
  • 분류와 회귀의 차이
    • 출력유형 : 분류는 레이블, 회귀는 연속적
    • 목적 : 분류는 결정경계 찾기(데이터를 설명하는 경계선), 회귀는 최대한 비슷한값찾기
    • 평가 : 분류는 정밀도사용, 회귀는 r2 에러사용
    • “How Much / How Many?” (회귀)
    • “Is this A or B?” (분류)
